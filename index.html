Plugin 1 - jQuery Validation Plugin

  • This plugin is used to assist in the validation of forms.
  • I am using this plugin to validate a form for buying bucks tickets.
  • This plugin is saves time by making form validation easier
  • https://jqueryvalidation.org

Plugin 2 - Invert Scroll

  • This plugin is used to invert the vertical scroll into a horizontal scroll.
  • I am using the scroll to make it appear as if Giannis is dunking.
  • This plugin can be used to create cool scrolling effects.
  • https://www.pixxelfactory.net/jInvertScroll/

Plugin 3 - jQuery Knob

  • This plugin is used to change text inputs into Dial graphics.
  • I used it to add a dial graphic to my page. The number can be changed by clicking the dial.
  • The plugin makes it extremely easy to customize a dial graphic to how you want it.
  • http://anthonyterrien.com/knob/

Plugin 4 - TwentyTwenty

  • This plugin is used to slide between two different images.
  • There are 2 JavaScript files and 2 css files that need to be used for this plugin to work.
  • Drag the slider the see the before and after images.
  • https://zurb.com/playground/twentytwenty

Plugin 5 - Warpdrive

  • This plugin is used to create a canvas with a warpdrive graphic effect.
  • I used it to create a interactive graphic that moves when the mouse is hovering over it.
  • When creating a new warpdrive canvas all you really need to do is set the settings then call the function.
  • https://github.com/NiklasKnaack/jquery-warpdrive-plugin
